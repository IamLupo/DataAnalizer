# DataAnalizer

This is a test project to extract data out of files.

## Clone Project

```
git submodule init
git submodule update

sudo apt-get install libpoppler-cpp-dev
```

## Compile project

For development:
```
cd DataAnalizer
mkdir build
cd build

cmake ..
make -j 4
```

## Run

For development:
```
cd DataAnalizer/build
./DataAnalizer
```
